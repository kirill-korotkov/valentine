# Сценарии и дизайн: Валентинка + Синхронизация

## Текущее состояние

- ✅ Карусель фото, вопрос, финалка, конфетти
- ✅ Главное меню, синхронизация (создать комнату / войти по ссылке)
- ✅ Счётчик «Вместе»
- ⚠️ Календарь, места, желания, фото — **только просмотр**, нельзя добавить

---

## Сценарии использования

### Сценарий 1: Первый запуск вдвоём

**Действия:**
1. Кирилл открывает приложение, проходит карусель → «Да» → конфетти.
2. Жмёт «Главное меню» → «Синхронизация» → «Создать комнату».
3. Копирует ссылку, отправляет в Telegram.
4. Партнёр открывает ссылку → попадает в ту же комнату.
5. Оба видят пустые разделы и могут добавлять данные.

**Что нужно:**
- Подсказка «Создай комнату и поделись ссылкой» при первом заходе в меню.
- Понятный пустой стейт: «Вместе добавьте первое желание», а не «Добавьте в разработке».

---

### Сценарий 2: Список желаний

**Действия:**
1. Оба заходят в «Список желаний».
2. Кирилл добавляет «Съездить в Грузию».
3. Партнёр видит через 3 сек и добавляет «Попробовать суши».
4. Кирилл дарит поездку → отмечает «Грузия» как «Сделано».
5. Партнёр видит галочку — желание исполнено.

**Дизайн:**
- Кнопка «+ Добавить желание» внизу или в header.
- Модалка / плашка: поле ввода + «Сохранить».
- Чекбокс по тапу — отметить выполненным.
- Для выполненных — зачёркнутый текст, приглушённый фон.

---

### Сценарий 3: Календарь дат

**Действия:**
1. Добавляют даты: знакомства, первого поцелуя, годовщины.
2. Открывают раздел — видят список по датам.
3. Партнёр добавляет забытую дату — она появляется у обоих.

**Дизайн:**
- Кнопка «+ Важная дата».
- Форма: выбор даты (date picker) + название + заметка.
- Список: дата крупно, название, заметка мелко.
- Сортировка от новых к старым или наоборот.

---

### Сценарий 4: Наши места

**Действия:**
1. Добавляют кафе, город, маршрут.
2. Каждый при необходимости дополняет список.
3. Планируют поездку по «Нашим местам».

**Дизайн:**
- Кнопка «+ Добавить место».
- Форма: название + заметка (опционально).
- Карточки: название, заметка, иконка места.

---

### Сценарий 5: Совместные фото

**Действия:**
1. Кирилл добавляет фото с телефона.
2. Оно загружается на бэкенд, появляется в галерее.
3. Партнёр открывает раздел — видит фото.
4. Тап по фото — полноэкранный просмотр.

**Дизайн:**
- Кнопка «+» в углу сетки.
- Выбор: камера или галерея.
- Сетка 2 колонки, превью.
- Тап по фото — fullscreen, свайп влево/вправо, закрытие свайпом вниз.

## Варианты дизайна

### Вариант A: Минималистичный

- Бежевый фон `#fef7f3`, белые карточки, красные акценты.
- Много воздуха, мало элементов.
- Кнопки «+» в правом нижнем углу экрана (FAB).
- Модалки с простыми формами.

### Вариант B: Карточный

- Крупные карточки с тенью и скруглениями.
- Пустой стейт — большая иллюстрация/иконка + текст + кнопка.
- Добавление через отдельный экран, не модалку.

### Вариант C: Тёплый / «конфетный»

- Пастельные оттенки: розовый, персиковый, мятный.
- Лёгкие градиенты, сердечки, мягкие тени.
- Более «подарочный» визуал.

## Конкретные UI-решения

### Желания
- FAB «+» или кнопка «Добавить желание» под заголовком.
- Модалка: `<input placeholder="Например: Съездить в Париж">` + «Готово».
- Список: тап по чекбоксу = toggle done. Долгий тап = удалить (с подтверждением).

### Календарь
- Кнопка «+ Важная дата».
- Форма: `<input type="date">` + `<input placeholder="Название">` + `<textarea placeholder="Заметка">`.
- Список: `formatDate(date)` + title + note.

### Места
- Кнопка «+ Место».
- Форма: название (обязательно) + заметка.
- Карточки как сейчас, плюс иконка MapPin.

### Фото
- Кнопка «+» в сетке.
- `<input type="file" accept="image/*">` — для веба.
- Для Capacitor — `@capacitor/camera` или `CapacitorFilesystem` + выбор из галереи.
- После выбора: конвертация в base64 → `addPhotoToCloud` → обновление gallery.
