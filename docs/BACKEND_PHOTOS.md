# Бэкенд: раздача фото

Фронтенд запрашивает фото по URL:
```
GET https://backend-for-render-valentine.onrender.com/api/photos/{roomId}/{photoId}
```

Бэкенд `backend-for-render-valentine` уже имеет этот маршрут. Проверь CORS для:
- `https://kirill-korotkov.github.io`
- `capacitor://localhost` (для iOS)
