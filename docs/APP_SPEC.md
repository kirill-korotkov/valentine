# Спецификация приложения Валентинка

## Поток приложения

1. **Вступление** (уже есть) — карусель → вопрос → финал с конфетти
2. **Главное меню** — экран с карточками разделов
3. **Разделы** — отдельные экраны по выбору

---

## Разделы (MVP)

### 1. Совместные фото
- Галерея с совместными фото
- Листание свайпом / стрелками
- **Добавление фото** — кнопка «Добавить», загрузка с устройства

### 2. Календарь дат
- Календарь с отмеченными важными датами
- Нажатие на дату → экран с:
  - Скрин переписки (фото)
  - Заметки / важные моменты

### 3. Счётчик «Вместе»
- Количество дней/месяцев/лет вместе
- Прогресс до следующей годовщины
- Настройка даты начала отношений

### 4. Наши места
- Список или карта мест (кафе, города, маршруты)
- Каждое место: название, фото, заметка
- Добавление новых мест

### 5. Список желаний
- Bucket list для двоих
- Добавление пунктов
- Отметка «Сделано»

---

## В разработке позже

- **Квиз «Насколько мы знаем друг друга»** — вопросы друг о друге, проверка ответов

---

## Технические решения

| Фича | Хранение данных |
|------|-----------------|
| Фото, даты, места, желания | Пока **localStorage** или **IndexedDB** (всё на устройстве) |
| Или | Бэкенд (база данных) — если нужна синхронизация между устройствами |

Для MVP — localStorage/IndexedDB, без бэка. Данные привязаны к устройству.
